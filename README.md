# 🛒 쇼핑몰 웹사이트

Node.js와 Express를 활용한 쇼핑몰 웹사이트입니다. 
사용자 인증, 상품 검색, 장바구니, 주문 관리 등 주요 기능을 포함합니다.

---

## 🛠 기술 스택
- **프론트엔드**: HTML, CSS, JavaScript
- **백엔드**: Node.js, Express
- **데이터베이스**: MYSQL
- **배포**: localhost

---

## 🚀 주요 기능
- 사용자 인증 (로그인/회원가입)
- 상품 조회 및 검색
- 장바구니 추가/관리
- 주문 관리 및 확인
- 전체 테이블(db(=) 관리
- 게시판 추가/수정/삭제 관리
